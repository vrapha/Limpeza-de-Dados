# Data Cleaning Portfolio (Python)

Este repositório contém meus projetos de estudo em **limpeza e preparação de dados**, realizados com Python e bibliotecas como Pandas e Matplotlib.

## Projetos incluídos

### Projeto 1 — Limpeza de Dados de Vendas
Pasta: `project-data-cleaning`

- Leitura de dados brutos (CSV)
- Tratamento de valores nulos
- Remoção de duplicatas
- Correção de preços inválidos
- Criação de coluna de valor total
- Geração de CSV limpo pronto para análise
- Visualização simples com gráfico de faturamento por produto

Este projeto representa meu nível atual (iniciante/intermediário) na área de análise de dados.
